# Favorites — Page Design Brief

Per-user starred entities across plugins.

## Pages

| Path | Archetype | Purpose |
|---|---|---|
| `/favorites` | Smart List | All starred items |

## Highlights

- Filter by entity type, plugin, last-touched
- Group by type
- Quick-unstar
- Cmd-K palette surfaces favorites first

This plugin contributes a star button via the rail across every Workspace Hub and Detail-Rich page.

## Cross-plugin

- All plugins — one-line API to register entities as starable
- `awesome-search-core` — boost favorites in search ranking
